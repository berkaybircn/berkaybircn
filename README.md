# 🚀 Featured Project

## Heyrexa

**Heyrexa** is an AI assistant platform designed to help individuals and businesses automate their daily workflows with intelligent AI agents.

### Key Features

- 🤖 AI Agents
- 💬 Telegram AI Assistant
- 📧 Gmail Integration
- 📅 Google Calendar
- 📁 Google Drive
- 🔎 Web Search
- 🧠 Claude & Gemini Models
- ⚡ Fast & Scalable Architecture

**Tech Stack**

`TypeScript` • `Next.js` • `Node.js` • `Supabase` • `PostgreSQL` • `Docker`
